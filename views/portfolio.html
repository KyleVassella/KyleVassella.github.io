Project Description:

This portfolio site (http://kylevassella.github.io) features a jQuery offset parallax hero image, jQuery smoothscroll nav buttons, and is a single page application powered by Angular 1. It's a work in progress and has gone through many changes in the past (feel free to browse through my old GitHub commits and see how bad it used to be). This site is 'hosted' for free as a gh-pages site via GitHub. As such, certain back-end tasks - like using PHP to send emails via a form submit button - are not readily possible.
