Project Description:

A website I created for a team of local Houston real estate agents. They wanted more web presence and thus required a dedicated website. The site features a Bootstrap inspired, jQuery-powered CMA (home valuation) form, Bootstrap grid style thumbnails for ease of responsive display, and a 'listings' page complete with iframe IDX property search. Eventually we will do without the iframe (these aren't readily found by search engine crawlers) and pay for a premium embeddable IDX plugin. Customer reviews are dynamically displayed on the home page via jQuery plugin (Slick Carousel). Additional Bootstrap-powered features - such as a contact form and mobile dropdown menu - are next on the to-do list.
