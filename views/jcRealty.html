J&C Realty

Freelance Web Design Project


Tools Used:

HTML5, CSS3, JavaScript (vanilla)

Github repository:

https://github.com/KyleVassella/JC-Realty

Project Description:

An ongoing project created for a team of local real estate agents. They wanted more web presence and thus required a dedicated website. Much work is still to be done, including a color scheme change and a layout overhaul. Note the 'Listings' page - complete with IDX property search i-frame. Eventually we will do without the i-frame (these aren't typically found by search engine crawlers) and pay for a premium embeddable IDX plugin. I'm currently working on improvements to the contact forms and the overall look of the site (much needed). Next on the list will be a way to dynamically display customer reviews on the home page - one of the team's most desired features.
